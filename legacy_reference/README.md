# Legacy reference

Código del proyecto anterior (**Anime Recommender**), conservado únicamente como
referencia visual/estructural para la reescritura de Swayp:

- `frontend/anime_recommender_app/lib/` — pantallas y servicios Flutter (login,
  recomendaciones, blacklist), referencia para la Fase 3 del roadmap
  (`docs/TODO.md`), cuando se reescriba el frontend genérico multi-dominio.
- `notebooks/` — notebooks de análisis exploratorio del dataset de anime.

Este código **no se ejecuta ni se mantiene**. No forma parte del pipeline actual
de Swayp y puede depender de módulos ya eliminados del repo (p. ej.
`python_runner.dart`, el backend Flask anterior). No lo actualices ni intentes
hacerlo compilar; consúltalo solo como punto de partida de diseño/UX.
